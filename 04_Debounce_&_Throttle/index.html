Debounce Explanation :
  • Debounce is a technique to limit the frequency of function execution, especially when dealing with events that might trigger rapidly, like typing in a search bar or button clicks
  • In the script file you can see EventListener for input field, that can we trigger a function call whenever a single character is typed, by that time if a request call is made for single character that will leads to performance issue
  • To overcome this we can use Debounce with a time delay after typing.

Throttle Explanation :
  • Throttling in JavaScript is a technique used to limit the rate of a function execution.
  • It ensures a function is called at most once in a specified time interval.
  • This is useful for improving performance and preventing excessive function calls, especially when dealing with events that fire rapidly, such as scrolling, resizing, or mouse movement.
